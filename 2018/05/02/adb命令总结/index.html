adb命令总结

Posted on 2018-05-02 Edited on 2018-11-28 In android

adb基本语法

$ adb [-d|-e|-s ]

参数含义:
|参数|含义|
|——|——|
|-d|指定当前唯一通过 USB 连接的 Android 设备为命令目标|
|-e|指定当前唯一运行的模拟器为命令目标|
|-s|指定相应 serialNumber 号的设备/模拟器为命令目标|

  2. 将设备与电脑通过 USB 线连接。

参数：
|参数|显示列表|
|——|——|
|无|所有应用|
|-f|显示应用关联的 apk 文件|

|-e|只显示 enabled 的应用|
|-s|只显示系统应用|
|-3|只显示第三方应用|
